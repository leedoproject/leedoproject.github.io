About LEEDO Project

리도 프로젝트는 커뮤니티를 중심으로 리도 NFT와 LEEDO 토큰을
탈중앙화 된 프로젝트의 구축을 목표로 합니다.
리도 프로젝트의 모든 결정은 커뮤니티와 DAO에 의해 결정되며
리도 프로젝트는 블록체인 게임 생태계의 핵심 구성요소인
게임, 게임 디파이, 커뮤니티, 마켓 등을 제공하는 개발팀에 대해
적합한 기술적 환경과 펀드를 공급하고자 합니다.
커뮤니티를 기반으로 새로운 프로젝트나 개발팀이 발굴되고
재정적으로 뒷받침되어 탈중앙화 된 게임 생태계를 만듭니다.

LEEDO 토큰

LEEDO 토큰은 리도 프로젝트에서 사용되는 화폐로 ERC-20 규격을
LEEDO 토큰은 체크인 한 사용자들에게 분배되며
첫 번째 시즌에는 7억 개가 발행되며 두 번째 시즌에는 3억 개가 발행됩니다.
DAO의 결정에 의해 추가 발행될 수도 있습니다.

리도 토큰의 용도는 커뮤니티에서 만들어가는 것이지만
저희가 예상하는 용도는 다음과 같습니다.

  1. 리도 프로젝트 내의 화폐로 사용됩니다
  2. 리도 월드 내에 인프라들이 들어오고 다양한 콘텐츠들이 공급되기 위해 사용되는 커뮤니티 펀드로도 사용이 됩니다.
  3. 리도 프로젝트에서 활동을 하는 사용자에게 리도 토큰이 주어집니다.
  4. 리도 토큰의 유동성 공급(예: 유니 스왑)을 위한 유동성 공급 펀드로도 사용됩니다.

FAQ

메타마스크 지갑에 Matic Polygon Mainnet 추가하는 방법

  1. 1. Polygon의 메인넷을 추가하려면 네트워크 선택 드롭다운을 클릭한 다음 사용자 지정 RPC를 클릭합니다.

  2. 2. Matic Polygon Mainnet 네트워크 정보
    Network Name : Matic Polygon Mainnet
    New RPC URL : https://polygon-rpc.com/
    Chain ID : 137
    Currency Symbol : MATIC
    Block Explorer URL : https://polygonscan.com/

  3. 3. 정보를 추가했으면 저장을 클릭합니다. 이제 네트워크 드롭다운 목록에서 Polygon의 메인넷으로 직접 전환됩니다.
LEEDO NFT 카드는 무엇인가요?
LEEDO NFT 카드는 LEEDO World로 체크인할 때 필요한 초대장이며 리도 월드의 다양한 서비스를 사용할 때 필요한 LEEDO 토큰을 보상받습니다.
LEEDO NFT 카드를 어떻게 얻을 수 있나요?
Opensea 마켓플레이스에 등록한 카드를 구매해 얻을 수 있습니다.
LEEDO NFT 카드를 구매했는데 어디서 그 카드를 볼 수 있나요?
LEEDO NFT 카드는 Opensea 마켓플레이스에 등록되어 있으므로 Opeansea 계정의 프로필에서 확인할 수 있습니다.
LEEDO NFT 카드는 어디에 쓰는 건가요?
LEEDO World로 체크인할 시 보상을 받을 수 있으며 LEEDO World 내에서 주민번호 같은 유니크한 ID로 사용되기도 합니다. LEEDO World에서 게임이 출시되는 경우 게임 캐릭터나 아이템으로 변환되어 사용될 수도 있습니다.
LEEDO NFT 카드로 게임이나 커뮤니티를 만들 수 있나요?
물론입니다. 누구나 LEEDO NFT 카드 각각의 고유한 속성을 사용해서 게임이나 커뮤니티를 만들 수 있습니다. 게임이나 커뮤니티를 만들 시 특정 환경이 필요한 경우 디스코드 채널로 들어오셔서 DAO에게 요청할 수 있습니다.
LEEDO NFT 카드의 가치는 어떻게 결정되나요?
LEEDO NFT 카드의 가치는 내재된 것이 아니라 LEEDO World에서 만들어지는 게임이나 커뮤니티 또는 서비스 등에서 부여됩니다.
